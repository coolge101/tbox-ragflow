# 在 Cursor 中打开本工程

1. 菜单 **File → Open Folder…**（或 **文件 → 打开文件夹**）
2. 选择本仓库根目录：

```text
tbox-ragflow-platform/
```

在本机克隆路径示例：

```text
/home/vboxuser/ragflow/tbox-ragflow-platform
```

打开后，工作区根即为平台根：`upstream/`、`apps/`、`deploy/` 等并列可见。

## 多根工作区（可选）

若需同时编辑「平台」与「仅上游」子树，可在 Cursor 中 **Add Folder to Workspace**，把 `upstream/ragflow` 再加进来（注意避免重复索引大目录）。
